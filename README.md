# Forum API

## Description
Backend API for a discussion forum with support for nested and threaded comments.

## Features
- Authentication (JWT)
- Threads CRUD
- Nested Comments
- Voting System (threads & comments)
- Admin Moderation

## Installation

npm install

## Usage

node src/server.js

## Technologies Used
Node.js  
Express.js  
MongoDB
